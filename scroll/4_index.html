Animation on scroll

해당 JavaScript 코드는 스크롤 이벤트를 감지하여 화면에 특정 박스 요소들을 보여주거나 숨기는 역할을 합니다.
먼저, const boxes = document.querySelectorAll('.box')는 CSS 클래스 이름이 "box"인 모든 요소들을 선택하여 boxes 변수에 저장합니다.
이후,window.addEventListener('scroll', checkBoxes)를 사용하여 스크롤 이벤트가 발생할 때마다 checkBoxes 함수가 실행되도록 등록합니다.
그리고 checkBoxes() 함수를 호출하여 초기 상태에서도 박스 요소들의 가시성을 설정합니다.
checkBoxes() 함수 내부에서는 화면의 높이(window.innerHeight)의 4/5 지점을 triggerBottom 변수에 저장합니다. 이 값을 기준으로 각 박스 요소의 위치를 계산하고,
해당 위치가 triggerBottom보다 작으면 "show" 클래스를 추가하여 박스를 보여주고, 그렇지 않으면 "show" 클래스를 제거하여 박스를 숨깁니다.
즉, 스크롤 이벤트가 발생할 때마다 박스 요소들의 위치를 계산하여 화면에 보여주거나 숨기는 것입니다. 이를 통해 스크롤에 따라 박스 요소들이 나타나거나 사라지는 효과를 줄 수 있습니다. 추가로 궁금한 점이 있으면 알려주세요.
